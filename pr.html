Adobe Premiere Pro Beginner Training Module

Note
For Editing Trainees (No Prior Editing Experience Required)

MODULE OVERVIEW

This training module is designed for absolute beginners who have no experience using Adobe Premiere Pro. By the end of this module, trainees will be able to confidently open Premiere Pro, import footage, perform basic edits, use essential keyboard shortcuts, and prepare projects for editing work.

This module focuses on clear explanations, step-by-step guidance, and hands-on practice suitable for wedding editors.

LEARNING OBJECTIVES

Define coding standards, architectural rules, or response tone. Keep instructions concise and actionable.

By the end of this module, trainees will be able to:

  • Understand what Adobe Premiere Pro is and how it is used in editing
  • Navigate the Premiere Pro interface
  • Import and organize media correctly
  • Use basic editing tools and the timeline
  • Apply essential keyboard shortcuts for faster editing
  • Play, cut, trim, and move clips confidently
  • Follow beginner editing best practices

PART 1: INTRODUCTION TO ADOBE PREMIERE PRO

What is Adobe Premiere Pro?

Adobe Premiere Pro is a professional video editing software used to create videos such as:

  • Wedding highlight films
  • Wedding reels for social media
  • Ceremony videos
  • Wedding podcasts and interviews

Premiere Pro allows editors to:

  • Cut and arrange video clips
  • Sync and edit audio
  • Add music, titles, and simple effects
  • Export final videos for clients
Important for trainees:
Premiere Pro may look complex at first, but you only need to master the basics to start editing weddings properly.

PART 2: OPENING PREMIERE PRO & CREATING A PROJECT

Step 1: Open Premiere Pro

Launch Adobe Premiere Pro from your computer

Step 2: Create a New Project

  • Click New Project
  • Name your project using company standards (example):

    • 2026-01-12_Maria-John_EDIT_v01

  • Choose the correct project location (inside the wedding folder)
  • Click Create
    • Rule:
    Always save your project inside the correct wedding folder to avoid missing files.

    PART 3: PREMIERE PRO INTERFACE (BEGINNER FRIENDLY)

    Main Panels You Must Know

    1. Project Panel
  • This is where all your videos, audio, and images are stored
  • Think of this as your media library
    • 2. Source Monitor

    Used to preview clips before adding them to the timeline

    3. Timeline
  • This is where you build your video
  • Video goes on the top tracks (V1, V2, V3)
  • Audio goes on the bottom tracks (A1, A2, A3)
    • 4. Program Monitor
  • Shows what your final video looks like
    • 5. Tools Panel
  • Contains tools like Selection, Razor, Hand, and Zoom

    • PART 4: IMPORTING MEDIA (VERY IMPORTANT)

    Correct Way to Import Files

    Method 1: Drag and Drop
  • Drag video/audio files directly into the Project Panel
    • Method 2: Import Menu
  • Click File > Import
  • Select your files
  • Click Open
    • Important Rule:
    NEVER move or rename files after importing them into Premiere Pro.

    PART 5: BASIC TOOLS (BEGINNER TOOLS ONLY)

    1. Selection Tool (V)

  • Used to move clips
  • Used to select clips

    • 2. Razor Tool (C)

  • Used to cut clips into pieces

    • 3. Hand Tool (H)

  • Used to move around the timeline

    • 4. Zoom Tool (Z)

  • Used to zoom in and out of the timeline

    • PART 6: USING THE TIMELINE

    Adding Clips to the Timeline

  • Drag clips from the Project Panel to the Timeline

    • Playing the Timeline

  • Press Spacebar to play or pause

    • Cutting Clips

  • Move the playhead to the desired cut point
  • Select the Razor Tool (C)
  • Click on the clip to cut

    • Deleting Clips

  • Select the clip
  • Press Delete
    • Beginner Tip:
    Avoid leaving empty gaps in the timeline.

    PART 8: BASIC AUDIO & MUSIC (INTRO LEVEL)

    Adding Music

  • Import music into the Project Panel
  • Drag music to an audio track (A1 or A2)

    • Audio Safety Rules

  • Avoid red audio meters (this means audio is too loud)
  • Dialogue should be clear and not overpowering

    • PART 9: BEGINNER EDITING PRACTICE

    Practice Task 1

  • Import provided training footage
  • Create a 30‚Äì45 second simple wedding montage
  • Use only:
    • Cuts
    • Music
    • Clean trimming

    Practice Task 2

  • Practice keyboard shortcuts:
    • Play / Pause
    • Cut clips
    • Zoom timeline

    PART 10: COMMON BEGINNER MISTAKES TO AVOID

  • Forgetting to save work
  • Leaving blank gaps in the timeline
  • Using too many effects
  • Editing without listening to audio
  • Renaming files outside Premiere Pro
